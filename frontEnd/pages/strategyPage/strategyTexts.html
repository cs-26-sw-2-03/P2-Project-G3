1. Naked Single

    The Naked Single only occurs when a cell has only one possible candidate number left after eliminating all other numbers that are already present in its row, column, and 3x3 box. Since there is no other number that can be placed in that cell, the solution is immediately determined. The reason why it is called "naked" is because the correct number is "visible" directly in the cell, and there is no advanced pattern detection that is required.

  2. Hidden Single

    The Hidden Single is a cell with multiple candidates, but one of those candidates cannot be in any other cell in the row, column, or box. That means that the candidate is hidden among the other candidates. When the hidden solution is found, it is the solution to that cell. In most Sudoku puzzles, the first five to ten cells that get solved will be hidden singles. The way to spot a hidden single is different depending on if the candidates can be seen for the remaining unsolved cells or not.

  3. Locked Candidates (Pointing Pair/Triple)

    The Locked Candidates occur when a candidate in a 3x3 box is restricted to one row or column. If that were to occur, then the candidate can be removed from the rest of that row or column outside the box.

  4. Naked Pair

    The Naked Pair is when there are two cells in the same row, column, or box that have the same pair of candidates. This is a valuable strategy because these two candidates can only appear in these two cells. This means that they will not appear in any other cell in the respective row, column, or box.

  5. Hidden Pair

    A Hidden Pair occurs when two specific numbers show up in the same two cells within a unit. These cells might also have other possible numbers. Any other candidate in these two cells can be eliminated. This limitation comes from positional logic. This requires scanning by number instead of by cell.

  6. Naked Triple

    The Naked Triple occurs when three cells each have the same candidates. Those numbers need to fill those three cells in some order. They can be eliminated from other cells within the unit. This concept is based on Naked Pairs.

  7. Hidden Triple

    The Hidden Triple occurs when three cells all contain exactly the same three potential values. Other candidates may be present in those cells. They can be eliminated from other cells in the same unit. This logic expands on Naked Pairs. This strategy requires closer attention to detect possible candidates, than a Naked Triple.

  8. X-Wing

    X-Wing, also known as a 2x2, is a single candidate strategy where 2 of the same candidates are limited to the same rows or columns. There are different types of X-Wing.

    X-Wing Type 1 is asymmetric, that is, when the X-Wing can be seen through one set eitherof the rows or columns. There are base sets, which are rows or columns where the X-Wing exists. Then there are cover sets where there will be made eliminations.

    X-Wing Type 2 is symmetric, that is, when the 2 X-Wings make the same eliminations, they are symmetrical.

  9. Simple Coloring

    The Simple Coloring tracks that one candidate number uses two alternative colours across the grid. It is based on strong links, which means that when a candidate appears twice in a row, column, or box, then one of those cells must contain the number. Simply put, by coloring these linked cells alternatively, chains of possibilities are created.

  10. XY-Wing

    The XY-Wing strategy involves the three cells that are connected through shared candidates. One pivot cell contains two candidates. Two wing cells each have one candidate in common with the pivot. This creates a forced elimination.

  11. XYZ-Wing

    The XYZ-Wing is similar to XY-Wing, but the difference is that it features a pivot that has three candidates instead of two. Two other cells restrict the placement of the candidate. The common candidate can be eliminated from the cells that overlap. The XYZ-Wing is a bit more complicated than XY-Wing.

  12. W-Wing

    The W-Wing connects two equivalent bi-value cells through a strong connection elsewhere. A linking possibility imposes logical limitations. This allows for the elimination of a shared candidate. What it does is that it combines pair logic with chain reasoning.

  13. Swordfish

    The Swordfish is a variation of the X-Wing strategy that uses three rows and columns. A number shows up in the same column placement across three separate rows. This creates a form of logical elimination pattern. The potential candidate can be eliminated from other cells in these columns. This involves reviewing several units.

  14. Alternating Inference Chains (AIC)

    The AIC links candidates together through a chain of strong and weak logical connections. This means that if there is a strong link, one candidate is false and the other must be true. If there is a weak link, then both candidates cannot be true at the same time.

  15. Forcing Chains

    The Forcing Chain assumes that there is one candidate and then it follows the logical consequences. If contradictions occur, then the candidate will be eliminated. This strategy is structured; it is done without guessing.

  16. Almost Locked Sets (ALS)

    The ALS refers to a group of cells that have n + 1 candidates spread over n cells. One of these candidates needs to be ruled out based on logical reasoning. ALS can connect with different patterns. They allow for complex eliminations.

  17. Sue de Coq

    The Sue de Coq integrates the interactions of rows, columns, and boxes. This creates a complicated relationship of subsets. Candidates can be eliminated from units that overlap.

  18. Jellyfish

    The Jellyfish expands the concept of swordfish to grid of four rows and four columns. A candidate positions itself in a structured pattern. This creates elimination opportunities in intersecting columns or rows.
