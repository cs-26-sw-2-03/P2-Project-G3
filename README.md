# P2-Project-G3
# Sudoku website with difficulty control
Created by students from Aalborg University (AAU) <br />
Student group CS-26-SW-2-03<br />

## Overview
This project is a website where users can play Sudoku puzzles with automatic difficulty adjustments.<br />
The website also includes a web page with strategies to solving puzzles. <br />
Specific Sudokus are recommended to users depending on their proficiency score which is calculated based on error rate and completion time.<br />
<br />
To run the program you must have node.js installed, then write the following in the terminal: node server.js<br />
<br />

## Usage
Users will complete Sudoku puzzles on the Sudoku web page of the website. <br />
Completing the Sudokus fast and without errors increases the difficulty of the next given Sudoku, while taking a long time or making many errors will decrease the difficulty. <br />
While solving these Sudokus, solving strategies will be recommended to the user. <br />
Learning these strategies and using them will increase the users ability to solve Sudokus. <br />
